# ListItemCache

Extends: `CacheItem`

## Properties

| Name | Type | Description |
|------|------|-------------|
| `id` | `string \| undefined` | The block ID of this list item, if defined. |
| `task` | `string \| undefined` | A single character indicating the checked status of a task. |
| `parent` | `number` | Line number of the parent list item (position.start.line). |
